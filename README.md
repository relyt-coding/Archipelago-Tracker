# Archipelago Tracker – Wind Waker Edition

A Wind Waker sea-chart themed check tracker for Archipelago multiworld randomizer races. Built for OBS Browser Source capture at 1920×1080.

## Games Tracked
- Majora's Mask
- Ocarina of Time
- Wind Waker

## Modes
- **Footer** — parchment strip along the bottom 10% of the screen
- **Compact** — 400×300 floating overlay, top-left corner
- **Full Screen** — large aesthetic tracker for BRB / breath screens

## OBS Setup
1. Add a **Browser Source**
2. Set URL to your Netlify URL (e.g. `https://your-site.netlify.app`)
3. Set width to **1920** and height to **1080**
4. The admin panel lives below the 1080px line — it won't appear in your stream capture

## Deployment
Deployed via GitHub → Netlify continuous deployment.

## Local Use
Just open `index.html` directly in a browser — no build step required.
